A Basic Rails ToDo List
------------------------

Purpose
========
This is a learning project for NSS Part 2.



Usage
======

This is a standard Rails app.  To run the tests, use 'rake'.  To run the app, use `rails s`



Features
=========

1. View tasks
2. Edit tasks
3. Multiple lists
4. Reorder tasks
5. Finish tasks
